## 使用方法 | How to Use

### 生成漫画图像 | Generate Comic Images

由于当前环境未配置图像生成 API 密钥，请使用以下方法生成图像：

#### 方法 1：使用 baoyu-image-gen 技能

```bash
# 1. 生成角色参考表
npx -y bun ./.skills/media/baoyu-image-gen/scripts/main.ts \
  --promptfiles ./comic/middleeast-conflict/characters/characters-prompt.md \
  --image ./comic/middleeast-conflict/characters/characters.png \
  --ar 4:3 \
  --provider openrouter

# 2. 生成封面
npx -y bun ./.skills/media/baoyu-image-gen/scripts/main.ts \
  --promptfiles ./comic/middleeast-conflict/prompts/00-cover-middleeast-storm.md \
  --image ./comic/middleeast-conflict/00-cover-middleeast-storm.png \
  --ar 3:4 \
  --provider openrouter

# 3. 批量生成内页（创建 batch.json）
# 见下文 batch.json 示例
```

#### 方法 2：使用批量模式 | Batch Mode

创建 `comic/middleeast-conflict/batch.json`:

```json
{
  "jobs": 4,
  "tasks": [
    {
      "id": "cover",
      "promptFiles": ["prompts/00-cover-middleeast-storm.md"],
      "image": "00-cover-middleeast-storm.png",
      "ar": "3:4"
    },
    {
      "id": "page-01",
      "promptFiles": ["prompts/01-page-historical-roots.md"],
      "image": "01-page-historical-roots.png",
      "ar": "3:4"
    }
    // ... 添加更多页面
  ]
}
```

然后运行：
```bash
npx -y bun ./.skills/media/baoyu-image-gen/scripts/main.ts \
  --batchfile ./comic/middleeast-conflict/batch.json \
  --jobs 4
```

### 合并为 PDF | Merge to PDF

生成所有图像后，使用 baoyu-comic 技能合并：

```bash
npx -y bun ./.skills/media/baoyu-comic/scripts/merge-to-pdf.ts \
  ./comic/middleeast-conflict/
```
