# Character Definitions - Middle East Conflict Comic
# 角色定义 - 中东冲突漫画

**Style**: manga (日式漫画)
**Art Direction**: dramatic (戏剧张力)
**Note**: This is a geopolitical educational comic using symbolic character designs.
**注意**: 这是地缘政治教育漫画，使用象征性角色设计。

---

## Character 1: 美国 Uncle Sam (象征形象)

**Role**: Narrator / Major Player / 主要参与方
**Age**: Timeless / 永恒

**Appearance**:
- Face shape: Angular jaw, strong features / 棱角分明的下颌
- Hair: White, short, styled / 白色短发，整齐
- Eyes: Blue, sharp, calculating / 蓝色眼睛，锐利
- Build: Tall, powerful build / 高大强壮
- Distinguishing features: Goatee beard, star-spangled motifs / 山羊胡，星条旗元素

**Costume**:
- Default outfit: Dark suit with American flag pin, military uniform variant / 深色西装配美国国旗胸针，军装 variant
- Color palette: Navy blue (#002868), Red (#BF0A30), White (#FFFFFF)
- Accessories: Eagle emblem, globe icon / 鹰徽章，地球仪

**Expression Range**:
- Neutral: Serious, statesmanlike / 严肃，政治家风范
- Determined: Jaw set, intense gaze / 咬紧牙关，目光坚定
- Concerned: Furrowed brow, hand on chin / 皱眉，手托下巴
- Commanding: Pointing finger, authoritative / 指向前方，权威性

## Character 2: 以色列 Lion of Judah (象征形象)

**Role**: Major Player / Defensive Fighter / 防御战士
**Age**: Timeless / 永恒

**Appearance**:
- Face shape: Strong, determined features / 坚定有力的面部特征
- Hair: Dark brown, short, military cut / 深棕色短发，军人发型
- Eyes: Brown, vigilant, intense / 棕色眼睛，警惕而专注
- Build: Athletic, compact / 运动型，精干
- Distinguishing features: Star of David motif, scar (symbolizing historical trauma) / 大卫之星元素，伤疤（象征历史创伤）

**Costume**:
- Default outfit: Military fatigues with Israeli insignia, or suit for diplomatic scenes / 以色列军装或外交场合西装
- Color palette: Blue (#0038b8), White (#FFFFFF), Olive green (#6B8E23)
- Accessories: Dog tags, Star of David necklace / 军牌，大卫之星项链

**Expression Range**:
- Neutral: Alert, ready / 警惕，准备就绪
- Angry: Baring teeth, intense / 露齿，激烈
- Determined: Squared jaw, focused / 咬紧牙关，专注
- Anxious: Sweating, wide eyes / 流汗，眼睛睁大

## Character 3: 伊朗 Persian Lion (象征形象)

**Role**: Major Player / Revolutionary / 革命者
**Age**: Timeless / 永恒

**Appearance**:
- Face shape: Sharp, angular / 棱角分明
- Hair: Black, turban or military cap variant / 黑色头发，头巾或军帽 variant
- Eyes: Dark brown, intense, ideological / 深棕色眼睛，专注而意识形态化
- Build: Lean, wiry / 精瘦
- Distinguishing features: Turban (religious leader variant), military beret (IRGC variant) / 头巾（宗教领袖 variant），贝雷帽（革命卫队 variant）

**Costume**:
- Default outfit: Clerical robes (diplomatic) or IRGC uniform (military) / 教士袍（外交）或革命卫队制服（军事）
- Color palette: Green (#239F5F - Islamic flag), White (#FFFFFF), Black (#000000)
- Accessories: Prayer beads, nuclear symbol (controversial) / 念珠，核标志（争议性）

**Expression Range**:
- Neutral: Stoic, principled / 坚忍，有原则
- Defiant: Chin raised, challenging / 抬起下巴，挑衅
- Strategic: Slight smile, calculating / 微笑，算计
- Angry: Shouting, veins visible / 喊叫，青筋暴露

## Character 4: 平民 Civilians (群体形象)

**Role**: Victims / Witnesses / 受害者与见证者
**Age**: Various / 各年龄段

**Appearance**:
- Multiple variants representing different populations / 多个 variant 代表不同人群
- Israeli civilian: Modern dress, anxious expression / 以色列平民：现代服装，焦虑表情
- Iranian civilian: Traditional or modern dress, concerned / 伊朗平民：传统或现代服装，担忧
- Palestinian civilian: Keffiyeh, weary expression / 巴勒斯坦平民：头巾，疲惫表情
- Lebanese civilian: Modern dress, fearful / 黎巴嫩平民：现代服装，恐惧

**Costume**:
- Varied everyday clothing / 各式日常服装
- Color palette: Muted earth tones, reflecting hardship / 柔和的大地色调，反映苦难
- Accessories: Children, backpacks, emergency supplies / 儿童，背包，应急物资

**Expression Range**:
- Fearful: Wide eyes, covering children / 恐惧，保护儿童
- Weary: Hollow eyes, slumped posture / 疲惫，眼神空洞
- Angry: Shouting at leaders / 愤怒，向领导人喊叫
- Hopeful: Looking at peace symbols / 希望，望向和平标志

## Visual Consistency Notes

1. **Character proportions**: All personified nations are tall, heroic figures (8 heads tall)
2. **Color coding**: Each nation has distinct primary colors for instant recognition
3. **Symbols**: National symbols (eagle, Star of David, Islamic motifs) are subtle design elements, not overwhelming
4. **Civilians**: Drawn with more varied, realistic proportions to emphasize humanity
5. **Emotional range**: Leaders show calculated emotions; civilians show raw, immediate reactions
