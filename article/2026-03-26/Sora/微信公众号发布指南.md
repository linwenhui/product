## 🚀 发布步骤

### 步骤 1：登录微信公众号后台

```
https://mp.weixin.qq.com/
```

### 步骤 2：新建图文消息

1. 点击左侧菜单 **内容与互动** → **图文消息**
2. 点击 **新的创作** → **写留言**（或 **草稿箱** → **新的创作**）

### 步骤 3：复制文章内容

**方法 A：使用浏览器打开 HTML 文件复制（推荐）**

1. 在 Chrome/Edge 浏览器中打开文件：
   ```
   /cv5/linwh/ClaudeCode/claudecode_test/article/2026-03-26/Sora 倒下，国产 AI 视频能站起来吗-inline.html
   ```

2. 按 `Ctrl+A` (Mac: `Cmd+A`) 全选

3. 按 `Ctrl+C` (Mac: `Cmd+C`) 复制

4. 在公众号编辑器中按 `Ctrl+V` (Mac: `Cmd+V`) 粘贴

**方法 B：直接复制 HTML 代码**

1. 用文本编辑器打开 `Sora 倒下，国产 AI 视频能站起来吗.html`
2. 复制全部内容
3. 在公众号后台点击工具栏的 **HTML** 按钮（或 `</>` 图标）
4. 粘贴 HTML 代码
5. 点击确定渲染

### 步骤 4：设置封面图

1. 点击 **封面图** 区域
2. 选择 **从本地上传**
3. 上传文件：`/cv5/linwh/ClaudeCode/claudecode_test/article/2026-03-26/imgs/cover-sora.jpg`
4. 调整裁剪区域（建议 2.35:1 横图）
5. 点击确定

### 步骤 5：填写摘要

在 **摘要** 框中填入：

```
2026 年 3 月 25 日，OpenAI 突然宣布关停 Sora 视频生成服务。这款曾被誉为"AI 视频革命开创者"的产品，在燃烧了 100 亿美元后戛然而止。前一天还在和迪士尼开会的团队，第二天就收到了关停通知。Sora 之死，是 AI 视频泡沫的破灭，还是国产工具崛起的开始？
```

### 步骤 6：设置原创声明

1. 滑动到页面底部
2. 打开 **原创声明** 开关
3. 确认原创类型

### 步骤 7：设置其他选项

| 选项 | 建议设置 |
|------|----------|
| **赞赏** | ✅ 开启 |
| **评论区** | ✅ 开启 |
| **在看** | ✅ 开启 |
| **合集标签** | AI 前沿 / 科技评论 / 行业分析 |

### 步骤 8：预览与发布

1. 点击 **预览** 发送到手机预览效果
2. 检查图片、排版、链接
3. 确认无误后点击 **群发**
4. 扫码确认发送
